FONT DEPENDENCY
---------------

The build depends on the font AlteHaasGroteskBold.  This is needed at build time (not runtime on end user system).  
You must install AlteHaasGroteskBold for the game to build.


HOW TO INSTALL
--------------

Under Windows 7, just double click the file AlteHaasGroteskBold.ttf and the system will assume you want to install it.
